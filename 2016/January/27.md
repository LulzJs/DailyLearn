###Autoform

to get a multiline input you need to make sure you get a textarea. This can be done in the collection definition by:
```js
TestSchema = new SimpleSchema({
    content: {
        type: String,
        label: "Content",
        autoform: {
            rows: 10
         }
    }
});
```
